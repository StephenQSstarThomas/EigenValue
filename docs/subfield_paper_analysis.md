# CritPt 公开问题集小领域与相关论文分析

本报告整合了 4 个只读子代理对 70 个公开 CritPt test challenges 的分析。依据文件为：

- `repo/data/public_test_challenges/json/Challenge_*.json`
- `source/text/06-SI.tex`
- `source/main.bbl`
- `reference_index.csv`

重要说明：`06-SI.tex` 给出的是 test set 的总参考文献列表，不是逐题逐条绑定。因此下表中的“相关论文”是根据题名、题干关键词和 `main.bbl` 条目做的最近邻匹配；低置信或仅方法层面相关的条目已显式标出。

## 总体结构

这 70 题覆盖的是前沿物理研究中的“可验证小任务”，不是传统教材题。高密度方向包括：

- 凝聚态与量子多体：moire/FCI、FQH/任意子、Hubbard/Hatsugai-Kohmoto、Kitaev、PXP scar、拓扑晶体、量子几何。
- 高能、引力、宇宙学与 QCD：全息 Weyl anomaly、AdS/BCFT、inflation/torsion、SYK、LaMET/quasi-PDF、Schwinger-Keldysh hydrodynamics、CFT correlators。
- 量子信息、量子计算与量子基础：distributed sensing、Holevo information、tensor networks、inverse Hamiltonian、quantum capacity、contextuality、relative entropy contraction。
- AMO、光学与精密测量：HHG、optical binding、OPA、levitated optomechanics、Penning trap cavity shift、spin squeezing、Yb magic wavelengths、superresolution。
- 统计物理、非线性、流体、生物物理与天体探测：cell-size stochastic growth、RBC/Rayleigh-Darcy、autocatalytic cycles、Efimov physics、dark matter interferometer detection。
